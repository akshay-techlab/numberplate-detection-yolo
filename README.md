# Real-Time Number Plate Detection System 🚗

A computer vision project that detects vehicle number plates from images or real-time video using YOLO object detection and extracts the license plate text using Tesseract OCR.

## 📌 Features
- Real-time number plate detection
- License plate text extraction using OCR
- Image and video input support
- Deep learning-based object detection
- Automated vehicle identification

## 🛠 Technologies Used
- Python
- YOLO (Ultralytics)
- OpenCV
- Tesseract OCR
- NumPy
- Pandas
